## Dataset

This project uses the **HR Dataset Clean & Raw (2M Rows)**, a large synthetic HR dataset containing approximately 2 million employee records.

The dataset includes employee demographics, salaries, departments, performance ratings, experience levels, and other workforce-related attributes.

Dataset source:
https://www.kaggle.com/datasets/rashadalaa/hr-dataset-clean-and-raw-2m-rows

Due to file size limitations, the dataset is not included in this repository.
